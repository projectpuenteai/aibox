Welcome to Kolibri

This guide walks you through everyday use of Kolibri with your students.

It works fully offline — everything runs from the school's computer.

Start with the common tasks How do I sign in?

This guide is written for teachers who may be using Kolibri for the first time. Each section covers one classroom job, with numbered steps. Read it in order or jump straight to the section you need using the sidebar on the left or the search bar at the top.

Tip

If you are short on time, start with the First-day checklist and the Most common tasks section. That covers most of what you'll need in your first week.

What is Kolibri?

Kolibri is a learning platform that works without an internet connection. Students can watch videos, read documents, do exercises and take quizzes on the school's computers, even when there is no signal.

As a teacher — called a coach inside Kolibri — you can:

  • Create classes and organize your students.
  • Assign lessons with a chosen selection of materials (videos, readings, exercises).
  • Prepare quizzes to evaluate what they have learned.
  • Review each student's progress from the coach dashboard.
Note

This guide uses teacher and coach interchangeably. On screen, Kolibri may say Coach (English) or Tutor (Spanish), depending on the language of the device.

Kolibri is already installed and configured on the school's computers. You do not need to install anything or change technical settings. Your job is to teach; this guide helps with the day-to-day.

How to sign in

To use the coach features (create classes, assign lessons, check progress), you must sign in with your teacher account.

Step by step

  1. Open the browser (Chrome, Firefox, or the browser installed on the computer).
  2. If the Kolibri screen is already open, find the SIGN IN button. If not, ask the school's lead for the Kolibri address (usually something like http://192.168.x.x:8080).
  3. If the computer belongs to more than one facility, choose the facility your account belongs to first.
  4. Type your username and, if asked, your password.
  5. Click SIGN IN.
Forgot your password?

Ask your facility admin (the person in charge of Kolibri at your school) to reset it. See Troubleshooting below.

Sign out when you're done

At the end of class, sign out so students don't end up using your teacher account:

  1. Click the menu icon at the top (three lines ☰ or your name initial).
  2. Choose Sign out.

Account types (roles)

Kolibri has four kinds of accounts. As a teacher, you'll usually work with the first two.

Role What they can do Used by teachers?
Learner View materials, do exercises, complete quizzes. Yes — this is the account every student will have.
Coach (teacher) Everything a learner can, plus: create lessons, quizzes, groups, and check progress. Yes — this is your own account.
Facility admin Everything a coach can, plus: create and edit users and classes for the school. Sometimes — only if you also run the school's user list.
Super admin Changes device-level settings, imports channels, manages permissions. No. Reserved for the project's technical team.
Two kinds of coach

When you create a coach account, you must choose between:

  • Class coach: can only follow the classes assigned to them. The most common choice for a classroom teacher.
  • Facility coach: can follow every class in the school. Useful for academic coordinators or principals.
Important for this deployment

In our deployment, teachers are not super admins. If a screen asks you to change device settings, import channels, or edit permissions, stop and notify the project's technical team.

Create student accounts

Each student needs their own account so Kolibri can track their progress. To create students, you must sign in as facility admin.

What if I'm only a coach?

Coaches don't create users. If your account is coach-only, ask the facility admin to create your students' accounts.

Step by step

  1. Sign in and open the Facility menu in the sidebar.
  2. Open the Users tab.
  3. Click NEW USER.
  4. Fill in:
    • Full name of the student.
    • Username (short, no spaces, easy to remember).
    • Password (may be optional, depending on facility settings).
  5. For User type, choose Learner.
  6. Demographic data (birth year, gender) is optional.
  7. Click SAVE.
Classroom tip

Use consistent, easy usernames for kids — for example maria.lopez or maria6a. Keep usernames and passwords in a safe notebook until your students have them memorized.

Reset a student's password

  1. Go to Facility → Users.
  2. Next to the student's name, click OPTIONS and choose Reset password.
  3. Type the new password twice and click SAVE.

Create teacher / coach accounts

The procedure is very similar to creating students, but you pick the Coach role.

  1. Sign in as facility admin.
  2. Go to Facility → Users and click NEW USER.
  3. Fill in full name, username, and password.
  4. For User type pick Coach.
  5. An extra selector appears. For a classroom teacher, choose Class coach. For a school-wide coordinator, choose Facility coach.
  6. Click SAVE.
Best practice

Create teachers before students. That way, when you create classes, you can assign the right coach from the start.

Create a class

Classes group together the students who receive the same lessons and quizzes. They can match a grade, a section, or a project.

Who can create classes?

Only the facility admin. If you are coach-only, ask the admin to create the class and assign you as the coach.

Step by step

  1. Go to Facility → Classes.
  2. Click NEW CLASS.
  3. Type a class name. Suggestions:
    • 3rd grade — Math
    • Grade 6
    • Reading — afternoon shift
  4. Click SAVE.

The class shows up in the facility's class list. It still has no students or coach yet — those are added in the next steps.

Enroll students in a class

Once the class exists, you must enroll students so they receive the lessons.

  1. Go to Facility → Classes.
  2. Click the class name to open it.
  3. Click ENROLL LEARNERS.
  4. The list shows facility students who are not yet in this class. Use the search field if the list is long.
  5. Check the box next to each student you want to add.
  6. Click CONFIRM.

Assign a coach to the class

  1. On the same class page, find the Coaches section.
  2. Click ASSIGN COACHES.
  3. Check the teacher in charge and click CONFIRM.
Tip

A student can belong to more than one class (for example, their regular class plus an extra reading group). Just enroll them in each class separately.

Create groups inside a class

Groups let you split a class into smaller subgroups — for example, by reading level, project, or interest. You can then assign different materials to each group without having to split children into separate classes.

Create a group

  1. Sign in as coach and open the Coach dashboard.
  2. Pick your class and go to the Groups tab.
  3. Click NEW GROUP.
  4. Type a name — for example, Beginner reading, Advanced reading, Plants project.
  5. Click SAVE.

Add students to a group

  1. From the Groups tab, click the group's name.
  2. Click ENROLL LEARNERS.
  3. Check the students you want to add to the group.
  4. Click CONFIRM.
Example

If part of your 4th grade reads fluently and others are still learning, create two groups: Beginner reading and Advanced reading. When you assign a lesson, pick the relevant group instead of the whole class.

Assign lessons

A lesson is a bundle of materials (videos, documents, exercises) that you prepare and deliver to your students for a specific stretch of time.

Create a new lesson

  1. Sign in as coach and open the Coach dashboard.
  2. Pick your class and go to the Lessons tab.
  3. Click NEW LESSON.
  4. Fill in:
    • Title — for example, Equivalent fractions.
    • Description (optional).
    • Recipients — the whole class, a group, or individual students.
  5. Click SAVE CHANGES. The lesson exists but has no resources yet.

Add materials to the lesson

  1. Inside the lesson, click MANAGE RESOURCES.
  2. Pick a channel (the source of materials — for example, Khan Academy).
  3. Browse the folders until you find the resource you want.
  4. Use the Search if you know the topic or title.
  5. Check the box of a resource to add it. A confirmation appears at the bottom.
  6. When done, click SAVE & FINISH.

Make the lesson visible

A new lesson is hidden from students until you make it visible:

  1. In the lesson summary, find the Visible to learners switch.
  2. Turn it on so students can see and work on the lesson.
  3. When the work period ends, turn it off to hide the lesson again.
Tip

If two classes work on the same topic, you don't have to build the lesson twice. From the options menu (three dots), pick Copy lesson and choose another class or group.

Create quizzes

A quiz is an evaluation made of questions drawn from the channels' exercises. It measures what your students have actually learned.

Step by step

  1. Open the Coach dashboard, your class, and the Quizzes tab.
  2. Click NEW QUIZ → Create new quiz.
  3. Type a title — for example, Fractions test — week 3.
  4. Choose when learners will see their score:
    • After submitting: each learner sees their score as soon as they finish.
    • After the coach ends the quiz: no one sees their score until you close the quiz.
  5. Pick recipients (whole class, a group, or individual learners).
  6. Click + ADD QUESTIONS.
  7. Set how many questions the quiz should have (default is 10; more than 25 is not recommended).
  8. Select the exercises Kolibri should draw questions from, then click ADD (N) QUESTIONS.
  9. When you're ready, click SAVE & CLOSE.
  10. For students to start, click START QUIZ.
Important

You can change questions only before you start the quiz. Once started, you can still change the title, recipients, and section order — but not the questions.

End the quiz

When time is up or everyone has answered, open the quiz and click END QUIZ. After this, scores are final.

Check student progress

The coach dashboard shows at any moment how your students are doing with the lessons and quizzes you've assigned.

Overview (Class Home)

When you open a class, the Class Home tab shows three blocks:

  • Quizzes: how many students have started and completed each one.
  • Lessons: how many students are progressing, finished, or needing help.
  • Class activity: recent notifications (e.g., "Maria and 3 others started lesson X").

See progress per student

  1. Go to the Learners tab in the coach dashboard.
  2. You'll see a table per student with completed exercises, scores, and last activity.
  3. Click a student's name to see the detail: every lesson and quiz assigned to them and how they are doing.

Spot who needs help

Kolibri flags students who answer many questions incorrectly as “needs help”. In the Quizzes tab, opening a quiz reveals a Difficult questions sub-tab that lists the questions students got wrong most often. Use that list to revisit the trickiest concepts in class.

Print or export a report

In report views you'll see two small buttons:

  • Printer button: print or save the report as PDF.
  • Download button: export as CSV (opens in Excel or any spreadsheet).

Help students find content

Students reach materials from two places:

  • Classes: here they find the lessons and quizzes you assigned.
  • Library: here they can freely explore all available channels.

Show them how to open a lesson

  1. The student signs in with their username and password.
  2. On the home screen they'll see their class. They click its name.
  3. They see the list of Lessons and Quizzes you assigned.
  4. They tap or click a lesson to start with the first resource.

Browse the library

  1. From the side menu, the student opens Library.
  2. They pick a channel (for example, Primary math) and browse the folders.
  3. To search by topic, they type a word in the Keyword field and press Enter.
  4. They can filter results by activity type (video, reading, exercise), language, or level.

Activity types

  • Watch: videos.
  • Listen: audio.
  • Practice: exercises with correct answers.
  • Read: PDF or ebook documents.
  • Explore: interactive apps.
Tip

Remind students that exercises require 5 correct answers in a row to count as complete. If they miss one, the streak resets.

First-day checklist

Run through these before your first class with Kolibri:

Before class

  1. Confirm with the facility admin that your coach account is already created.
  2. Make sure your class is created and you appear as its coach.
  3. Make sure all your students are created and enrolled in the class.
  4. Try signing in on a classroom computer before the kids arrive.
  5. Prepare at least one lesson with two or three simple resources and turn its visibility on.

First class with students

  1. Hand out (or write on the board) the usernames and passwords.
  2. Show them how to sign in, step by step.
  3. Show them how to enter their class and open a lesson.
  4. Explain the basic icons: in progress, completed, bookmark.
  5. Remind students to sign out when they finish.
  6. After class, open the coach dashboard to see who progressed and who needs help.

Troubleshooting

Tap or click each problem to see the fix.

A student can't sign in
  1. Check that the student is typing the username exactly right (no extra capitals, no spaces).
  2. If they've forgotten the password, ask the facility admin to reset it (Facility → Users → Options → Reset password).
  3. Confirm the student is selecting the correct facility (when more than one is available).
  4. If it still fails, set a simple new password, let them in, then have them change it.
A student doesn't appear in my class
  1. Sign in as facility admin (or ask the admin to help).
  2. Go to Facility → Classes and open the class.
  3. Click ENROLL LEARNERS and check whether the student is in the list to enroll. If yes, tick the box and confirm.
  4. If they don't appear, go to Facility → Users and verify the account exists in this facility and is of type Learner.
  5. If the student belongs to a different facility, they won't show up here. Contact the technical team.
I can't assign lessons or quizzes
  1. Make sure you're signed in with your coach account, not a learner one.
  2. If you only see the Learn tab and not Coach, your account is probably set as learner. Ask the admin to change the user type to Coach.
  3. If your account is a Class coach, you'll only see classes assigned to you. Ask to be assigned to the relevant class.
Content for a subject is missing
  1. Make sure you're browsing Library, not just your class.
  2. Use the search field with topic keywords (e.g., fractions).
  3. Try filtering by language.
  4. If you confirm the channel you need is not installed, contact the project's technical team. Content is managed from the school's server and isn't something a teacher needs to install.
The page doesn't load or stays blank
  1. Check the Kolibri address is correct (e.g. http://192.168.x.x:8080).
  2. Refresh with F5 or the browser's reload button.
  3. Close and reopen the browser.
  4. If the problem happens on every computer, the Kolibri server may be down. Contact the technical team.
  5. If only one computer is affected, try restarting that computer.
I'm not sure which account type to use
  • Children in your class: Learner.
  • Classroom teachers: Class coach.
  • Academic coordinators or principals who need to see every class: Facility coach.
  • Person in charge of managing users and classes for the school: Facility admin.
  • Technical staff who maintain the server: Super admin. Teachers should not have this role.
A student finished the lesson but progress doesn't show
  1. Ask the student to close and reopen the lesson. There can be a small delay while Kolibri saves data.
  2. Refresh your own coach dashboard.
  3. If students use devices that sync to a central server, it's normal for progress to take a little time to appear. The View learner devices link on the class page shows when devices last synced.

Quick reference

Summary of the most-used steps. You can print this section as a one-page handout.

Sign in

  1. Open Kolibri in the browser.
  2. Pick the facility if asked.
  3. Type username and password.
  4. Click SIGN IN.

Create a student (admin)

  1. Facility → Users → NEW USER.
  2. Name, username, password.
  3. Type: Learner. SAVE.

Create a class (admin)

  1. Facility → Classes → NEW CLASS.
  2. Class name. SAVE.

Enroll students into a class

  1. Open the class.
  2. Click ENROLL LEARNERS.
  3. Tick the students. CONFIRM.

Create a group (coach)

  1. Coach → class → Groups → NEW GROUP.
  2. Name. SAVE.
  3. Open the group → ENROLL LEARNERS.

Assign a lesson

  1. Coach → class → Lessons → NEW LESSON.
  2. Title, description, recipients. SAVE CHANGES.
  3. MANAGE RESOURCES → choose materials → SAVE & FINISH.
  4. Turn on Visible to learners.

Create a quiz

  1. Coach → class → Quizzes → NEW QUIZ.
  2. Title, recipients, number of questions.
  3. Select exercises → ADD QUESTIONS → SAVE & CLOSE.
  4. START QUIZ to open it for students.

Check progress

  1. Coach → class → Class Home (overview).
  2. Learners tab for per-student view.
  3. Quizzes tab → pick a quiz → Difficult questions to see common errors.

Short glossary

TermMeaning
KolibriAn offline learning platform that runs on the school's computer.
FacilityThe organizational unit that groups users and classes (e.g., a school).
ChannelA collection of learning materials (e.g., Khan Academy videos).
ResourceA single material: a video, a reading, an exercise.
LessonA selection of resources a coach assigns to a class, group, or individual learners.
QuizAn evaluation built from exercise questions. Previously called Exam.
CoachThe teacher role: can create lessons, quizzes, and groups, and see progress.
Facility adminManages users and classes inside the facility.
Super adminManages the server and channels. Not a teacher's role.
SyncThe process by which devices share progress with a central server.
